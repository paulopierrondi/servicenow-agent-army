# Contributing

Contributions are welcome.

## Add an Agent

1. Create `agents/<id>.md`.
2. Add the entry to `catalog/agents.json`.
3. Run `npm run validate`.

## Add a Workflow

1. Create `workflows/<id>.json`.
2. Add the entry to `catalog/workflows.json`.
3. Run `npm run validate`.

## Standards

- Keep examples concrete.
- Do not invent ServiceNow APIs.
- Add guardrails for any write path.
- Keep beginner instructions clear.
- Use official docs when making factual claims.
