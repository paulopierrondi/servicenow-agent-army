# Prompt: Create an Agentic Workflow

```text
Use the Workflow Composer and Guardrails Reviewer.

Create an AI Agent Studio agentic workflow for:

<business outcome>

Constraints:
- Target AI Agent Studio guided setup.
- Do not assume undocumented deployment APIs.
- Include human approval for risky actions.
- Include tests for happy path, exception path, and permission failure.

Return:
1. workflow name
2. workflow description
3. trigger
4. variables
5. AI agents assigned
6. numbered steps
7. approval points
8. failure handling
9. audit notes
10. test cases
11. compact version for AI Agent Studio fields
```
